
Make sure `bottom-navigation` is defined before any footer plugin.

```json
{
  "plugins": [
    "bottom-navigation",
    "page-footer-ex"
  ]
}
```

You can also customize the border color with `color` property.

```json
{
  "pluginsConfig": {
    "bottom-navigation": {
      "color": "#3884FE"
    }
  }
}
```
